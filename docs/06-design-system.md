# Design System

The public site uses restrained industrial/medical styling with clear hierarchy,
strong content density, and accessible controls.

## Principles

- Prefer reusable layout helpers for spacing and width.
- Use existing UI primitives before introducing new visual patterns.
- Keep cards for repeated items, tools, and dialogs.
- Use lucide icons for common actions when an icon exists.

## Core Helpers

- `Container`: page-width constraint and responsive horizontal padding.
- `Section`: vertical spacing for public page sections.
- `SectionHeading`: eyebrow, title, and description grouping.
